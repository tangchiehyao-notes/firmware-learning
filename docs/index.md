# Introduction

這個網站先定位成單一入口，用來整理 OpenBMC / Embedded Linux 的學習地圖、知識骨架與各主題 repo。

目前我希望它先做好三件事：

1. 先提供清楚的 `Learning Roadmap`
2. 先整理每條主題線的入口頁
3. 讓未來的內容可以逐步回收到同一個網站

## 目前主導航

- Linux Internals
- OpenBMC Architecture
- Yocto
- Docker
- Firmware Debugging

## 暫時不放進主導航的主題

`Modern C++` 目前先保留在獨立 repo，之後等內容整理出比較完整的 introduction 與章節結構，再加回站內主導航。
