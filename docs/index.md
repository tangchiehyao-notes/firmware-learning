# Introduction

這個網站目前先定位成單一入口，用來整理 OpenBMC / Embedded Linux 的學習地圖、知識骨架與各主題 repo。

## 這個頁面的角色

- 提供整個網站的使用方式
- 說明目前有哪些主題已經進站內
- 說明哪些主題還先維持在獨立 repo

## 目前主導航

- Linux Internals
- OpenBMC Architecture
- Yocto
- Docker
- Firmware Debugging

## 暫時不放進主導航的主題
